# Customer Churn Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Telco Customer Churn dataset to understand customer behavior and identify factors that influence customer churn.

The analysis includes data cleaning, preprocessing, visualization, and correlation analysis to extract meaningful business insights.

## Dataset

Dataset Used: Telco Customer Churn Dataset

Features include:

* Customer Demographics
* Tenure
* Monthly Charges
* Total Charges
* Contract Type
* Internet Service
* Payment Method
* Churn Status

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Project Workflow

### 1. Data Loading

* Imported dataset using Pandas
* Examined dataset structure and columns

### 2. Data Cleaning

* Checked missing values
* Converted TotalCharges to numeric datatype
* Handled blank values
* Checked duplicate records

### 3. Exploratory Data Analysis

* Churn Distribution Analysis
* Histograms
* Boxplots
* Countplots
* Correlation Heatmap

### 4. Business Insights

Key observations from the analysis:

* Customers with lower tenure are more likely to churn.
* Higher monthly charges are associated with increased churn.
* Long-term customers show greater retention.
* Contract type significantly impacts churn behavior.
